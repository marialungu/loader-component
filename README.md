# loader-bounce

Simple loader component for react web apps


Installation
------

### Using npm:

`npm install loader-bounce`

Demo
------

![Demo of loader](https://github.com/marialungu/loader-component/blob/master/demo.gif)

Usage
------

```
import Loader from 'loader-bounce'

...

<Loader />
```

You can also configure the height, width and color of the loader:

```
import Loader from 'loader-bounce'

...

<Loader height={'90px'} width={'90px'} color={'#007A71'}/>
```

### Default values

| prop     | value   | type   |
| -------- | ------- | ------ |
| height   | 50px    | string |
| width    | 50px    | string |
| color    | #66064A | string |
